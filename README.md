qcd is a tool to quickly change directories. 
===
Install
---

>cp qcd.sh /etc/profile.d/

>cp qcd.py /usr/bin/

restart the terminal 

Use
---

>$ qcd -h

> qcd <数字> 切换目录
> qcd 切换到目录1或2
> \----------------
> -l 列出所有目录
> rm <path> 删除指定目录
> append <path> 追加指定目录
> -i  <path> 插入目录
> clear 清空目录
> -h 帮助
